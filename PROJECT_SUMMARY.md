🚀 NÄCHSTE SCHRITTE:

1️⃣  SETUP (Falls nicht geschehen)

   # Node.js installieren (falls nicht vorhanden)
   brew install node          # macOS
   # oder von https://nodejs.org/

2️⃣  PROJEKT STARTEN

   cd "/Users/S43DB/Desktop/isola verda"
   npm install                # Dependencies installieren
   npm run dev                # Development Server (http://localhost:5173)

3️⃣  ÄNDERUNGEN VORNEHMEN

   # Speisekarte ändern:
   → Bearbeite: src/data/menu.json

   # Restaurantinfos ändern:
   → Bearbeite: src/data/restaurant.json

   # Design anpassen:
   → Bearbeite: tailwind.config.js oder src/styles/globals.css

   # Seiten-Inhalte ändern:
   → Bearbeite: src/pages/*.vue Dateien

4️⃣  FÜR PRODUKTION BAUEN

   npm run build              # Production Build (dist/ folder)
   npm run preview            # Preview testen

5️⃣  DEPLOYEN

   → Siehe README.md für Deployment-Optionen:
   - Netlify (einfachste Option!)
   - Vercel
   - GitHub Pages
   - Docker
   - Eigener Server
